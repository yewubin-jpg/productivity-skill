# v2.6 Architecture: The Course Recommendation System

## I. Core Objective

To create a seamless bridge for users from **using** the tool to **learning** the methodology. This system will intelligently identify moments of high user satisfaction or deep curiosity and use them as opportunities to introduce YiXiaoNeng's official courses.

## II. System Architecture

The system will be implemented as a new engine, `course_engine.md`, which will be called by the `motivation_engine.md` after a successful interaction.

```
┌──────────────────────────┐
│   User Interaction       │
└──────────┬───────────────┘
           │
           ▼
┌──────────────────────────┐
│   Main Workflow          │
│ (Energy, Priority, etc.) │
└──────────┬───────────────┘
           │ Result + Context
           ▼
┌──────────────────────────┐
│   Motivation Engine      │ ← Delivers the primary response
└──────────┬───────────────┘
           │ User's Reply (e.g., "Thanks!", "How do I learn more?")
           ▼
┌──────────────────────────┐
│   Course Engine          │ ← Analyzes reply for trigger words
└──────────┬───────────────┘
           │
    ┌──────┴──────┐
    │             │
    ▼             ▼
┌───────────┐  ┌───────────────────┐
│ No Trigger│  │   Trigger Found   │
└───────────┘  └───────────────────┘
    │             │
    ▼             ▼
 (End)       ┌───────────────────┐
             │  Recommend Course │
             └───────────────────┘
```

## III. The `course_engine.md` Logic

This engine will contain a set of rules to detect trigger phrases in the user's reply.

### Trigger Categories:

1.  **Appreciation Triggers**: Keywords that signal high satisfaction and trust.
    *   **Keywords**: "太棒了", "谢谢你", "很有用", "解决了我的大问题", "amazing", "thank you", "so helpful".
    *   **Action**: Offer a gentle, value-add recommendation.

2.  **Deep-Dive Triggers**: Keywords that signal a desire for more systematic learning.
    *   **Keywords**: "怎么系统学习", "有没有课程", "我想深入了解", "how can I learn more", "is there a course".
    *   **Action**: Offer a direct, strong recommendation with clear calls to action.

### Recommendation Tiers:

The engine will provide tiered recommendations based on the trigger type.

*   **Tier 1 (Appreciation)**: A soft recommendation focusing on free resources to build further trust.
    *   **Content**: "很高兴能帮到您！如果您想了解更多叶武滨老师的原创方法，可以关注【易效能】的官方视频号和公众号，那里有大量免费的干货。"

*   **Tier 2 (Deep-Dive)**: A strong recommendation for paid courses, emphasizing efficiency and systematic learning.
    *   **Content**: "很高兴您有兴趣系统学习！要最快、最深入地掌握这套方法，参加叶武滨老师的官方课程是最佳途径。课程凝聚了15年的研究和实践精华，能帮您彻底建立高效能的习惯。"
    *   **Call to Action**: Provide both the official website and the direct registration link.
        *   **官网**: [www.yixiaoneng.com](https://www.yixiaoneng.com)
        *   **报名链接**: [http://yxndw.com/nbZYW5j](http://yxndw.com/nbZYW5j)

## IV. Integration with `motivation_engine.md`

The `motivation_engine` will be updated. After delivering its primary coaching message, it will pass the user's subsequent reply to the `course_engine` for analysis. This ensures that the course recommendation feels like a natural extension of the conversation, not a random advertisement.

## V. Final Goal

This system will create a complete user journey:

1.  **Attraction**: User finds the skill.
2.  **Activation**: User starts using the skill and finds it helpful.
3.  **Adoption**: User makes it a regular habit.
4.  **Advocacy/Ascension**: User is so satisfied they want to learn the core methodology, and the system seamlessly guides them to the official courses.
